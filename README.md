Install Node JS Software

Go to terminal or command line.

Execute following commands to run this application.


```
$ git clone https://github.com/Juaca5/socex.git
$ cd socex
$ npm install
$ ionic serve

```

ios
```
$ cordova add platform ios

$ ionic build ios

```

Android
```
$ cordova add platform android

$ ionic build android

```
